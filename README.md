# LanguageLearningApp
App for learning German/English/Russian vocabulary.

#Technical data  
Made for: Windows  
Resolution: 1920x1080  
Scaling: 100%

#Features:  
400+ Words in each language.  
Ability adding your own ones.  
Ability to blend one or multiple languages to check yourself.  
Ability to take random words to check yourself.  
Ability to choose range of words that are gonna appear in random selector (for example from word 50 to word 70).  
Search bar for words.  

#Controls  
Up => go one word up.  
Down => go one word down.
Scroll Up/Down for the same effect.  
Shift+R => redact line.  
Shift+Enter => add line. 
Backspace => delete line.
Shift + Left Mouse Button => put first clamp for setting range of random words (first you need to press the violet button with C).  
Shift + Right Mouse Button => same, but for second one (note that the first one can not stand after the second one and vice versa).  
Everything else is pretty intuitive.

#How to run the app  
Click the green Code button on this repository.  
Select Download ZIP.  
Extract the ZIP file to any folder.  
Open the WorkingApp folder.  
Run the .exe file.  

#Notes  
It is supposed to not be full-screen, it's made so that it's more comfortable to switch to browser tab to look up for definition of a word or something similar.
